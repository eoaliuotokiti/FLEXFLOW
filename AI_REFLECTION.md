My AI Reflection 
I made a workout app called FlexFlow. I used a program called Visual Studio Code to write the code. One thing I learned was how to use a virtual environment. This keeps my project files separate from other things on my computer. It makes sure the app has exactly what it needs to run.

I also learned how to use Git and GitHub. This was a way to save my work and put it online. I learned that I have to add my files first. Then I make a commit to save them. Finally I push the code to GitHub so others can see it. It was like making a save point in a video game. If I make a mistake I can go back to an old version of my code.

One problem I had was a mix up in the terminal. I tried to type a git command while my app was still asking me for a workout day. The app told me it was an invalid day. This taught me that the computer can only do one thing at a time in the terminal. I have to finish my app or close it before I can use git again.

Working with an AI was helpful. It was like having a teacher right next to me. It also showed me how to use a dictionary to store my workout plan. This made the code much shorter and easier to read.

In the future I want to make FlexFlow even better. I want to add a way for people to save their progress. I also want to add more exercises for each day. Learning these tools was a big step for me in becoming a programmer.
